# ⚠️ Your Supabase Project is Paused

## The Issue

You're seeing this error:
```
❌ Network Error: Failed to fetch
TypeError: Failed to fetch
```

**This means your Supabase project is paused or inactive.**

---

## ✅ How to Fix (2 Minutes)

### **Step 1: Open Supabase Dashboard**

Click this link:
```
https://supabase.com/dashboard/project/zqcspamakvfzvlqbunit
```

Or go to: https://supabase.com/dashboard

---

### **Step 2: Resume Your Project**

You'll see one of these:

**Option A: Banner at the top**
```
⚠️ Project is paused
[Resume Project] button
```
→ Click **"Resume Project"**

**Option B: Gray/inactive indicator**
```
Project Status: Inactive
[Restore] or [Resume] button
```
→ Click **"Resume"** or **"Restore"**

**Option C: Settings**
```
Go to: Settings → General
Look for: Project Status
Click: Resume Project
```

---

### **Step 3: Wait for Activation**

After clicking Resume:
- ⏱️ Wait 30-60 seconds
- 🟢 Status will change to "Active"
- ✅ Green indicator will appear

---

### **Step 4: Refresh Your App**

Once the project is active:
1. Go back to your application
2. Press `F5` or refresh the page
3. The error should disappear
4. You should be able to login

## 🔧 Why Does This Happen?

**Supabase Free Tier:**
- Projects pause after 7 days of inactivity
- Projects pause if you exceed limits
- Manual pause by user

**Solution:**
- Keep the project active by using it regularly
- Upgrade to a paid plan for always-active projects
- Resume manually when needed
